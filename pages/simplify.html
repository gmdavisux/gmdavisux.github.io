Simplified Install

Kodak Alaris | Lead Design | 2015

At Kodak Alaris, I tackled the challenge of simplifying the install process for high-speed scanning systems, reducing user friction for data processing teams. The original 57-step onboarding, packed with irrelevant data requests, needed a rethink to meet user needs efficiently.

Documenting & Refining the Current State

I started by unpacking a scanner and documenting its 57-screen install process, uncovering requests for addresses, employee numbers, and revenue. Digging deeper, I found marketing, sales, and support didn‘t use this data. By proving its redundancy, I convinced the team to eliminate it, streamlining the experience.

Benchmarking Best Practices

To guide the redesign, I studied top install processes. Standouts included:

Bose Speakers

Bose‘s app-based setup finds devices and connects them in 6 intuitive steps.

Google Chromecast

Chromecast uses the TV to direct users to an app, completing setup in a few simple steps.

Redesigned Workflow

Addressing diverse user environments—no CDs, limited internet—I proposed embedding stable software on the scanner. The workflow adapts to connection types:

  • The scanner‘s display guides users to a website or local network.
  • An ad-hoc network (inspired by benchmarking) supports wired or Wi-Fi setup.
  • With internet, it downloads the latest software; offline, it uses the shipped version.
  • Post-install, it auto-connects to nearby scanners, ready for use.

Results from Testing

Usability testing compared the old CD install to the new approach. Eight industry-experienced participants found the new version significantly faster to complete and overwhelmingly preferred it, proving a user-centered win.
