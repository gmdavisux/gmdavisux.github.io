Simplified install

Kodak Alaris | Lead Design | 2015

Kodak Alaris makes high speed scanning systems for data processing. I needed to simplify the install process.

Documenting the Current State

One of the first things I did at Kodak Alaris was to get myself an unopened scanner in a box and document the installation process. The onboarding process involved clicking thru 57 screens of info regarding adresses, phone, number of employees, revenue, and more.

I started to ask what the information was needed for. I was directed to marketing.

  • Marketing told me they dont use the info. Ask Sales.
  • Sales told me they don't use it. Ask Support.
  • Support assured me they get all their info on each phone call.

After I was able to demonstrate that the information was being collected and stored for nobody, I was able to convince the team we dodn't need it.

Benchmarking

Next I started looking for the best in class installation processes. There were a few that stood out.

Bose Speakers

Bose wireless speaker sytems use an app to find new devices, then easily add them to your network in about 6 steps.

Google Chromecast

Google Chromecast uses the TV screen to direct users to download an app (or visit an URL) needed to complete setup in a few simple steps.

The Design Work

With CDs becoming less common, and some government customers without inetrrnet, I proposed we include a stable version of the software install on the device. Using a trick I learned from benchmarking, we can create a local network via wire or wifi from the device. From there we can 'bootstrap' the connection to the device and download the software.

We had different versions dependingon the type of PC (wired or wirelss networking) and the type of scanner wired or wireless connection.
