Rescuing an out-of-scope design

We needed to turn an engineering prototype into a product with an aggressive schedule. Resource constraints threatened to push back design changes until after release. 

I was able to convince developers to allow me access to source code and make only CSS and HTML changes to transform the appearance without changing the existing back-end code.

Before & After a redesign using mostly CSS

We started with the engineering prototype above. I used mostly CSS to transform the appearance. I also implemented HTML and C++ changes when necessary.

Matching scanner control panel

Although I didn’t design the scanner or it’s control panel, the icons and colors of the app were designed to look good on the control panel.
