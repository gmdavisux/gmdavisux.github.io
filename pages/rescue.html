Design Rescue

Kodak Alaris | Lead Design, Research Planning and Facilitation | 2015

We needed to turn an engineering prototype into a product with an aggressive schedule. Resource constraints threatened to push back design changes until after release. 

I was able to convince developers to allow me access to source code and make only CSS and HTML changes to transform the appearance without changing the existing back-end code.

New Design

I was able to transform the appearance to a finished product. I used mostly CSS to transform the appearance. I also implemented HTML and C++ changes when necessary.

For testing and communication, I also created an interactive prototype using html, css, and angular.js Try it!

Legacy Design

We started with the engineering prototype. We developed a completely different design and workflow. All of this was out of scope for initial launch.

Matching scanner control panel

The icons and colors of the app were designed to look good on the control panel, which I knew was being redesigned. (I didn’t design the scanner or it’s control panel)

Companion mobile app

Keeping in mind the design strategy I established earlier, I also later designed the companion mobile app.
